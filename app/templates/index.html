¿Cómo puedo calcular la hipotenusa de un triángulo rectángulo?

Hace 1 minuto
Matemáticas → Geometría → Teorema de Pitágoras
Confianza: 95%

Para calcular la hipotenusa de un triángulo rectángulo, puedes usar el Teorema de Pitágoras:

a² + b² = c²

Donde 'c' es la hipotenusa

Si conoces los dos catetos (a y b), la hipotenusa será: c = √(a² + b²)
